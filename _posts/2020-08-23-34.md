# 문제/원인

- 이미 git에서 변화를 감지하고 있는 경우에는 .gitignore에 추가 하여도 계속 트래킹을 하게 됨

# 해결 방법

- 작업하던 내용이 있으면 모두 commit하고 진행할 것!
- 기존의 cache를 지우고 다시 add 해 줌

```bash
git rm -r --cached .
git add .
git commit -m "Fix untracked files"
git push origin master
```
